Sistema de Help Desk Produzido por um Dev Full Stack

Olá! Seja muito bem-vindo(a) ao Sistema Help Desk, desenvolvido por Alexandre Bonturi Nóbrega / Desenvolvedor Full Stack.
Esse é um sistema de gerenciamento de chamados de um Help Desk, desenvolvido em Java 11, Spring Boot 2.7.5, Angular 15.0.2, Node 18.12.1 e MySQL.
O projeto tem como objetivo principal abordar os conceitos de desenvolvimento de aplicações Web, Autenticação e Autorização de usuários com Tokens JWT, Spring Security, REST APIs, JPA e muito mais.
Em produção, o sistema estará rodando na nuvem sendo o front-end na Vercel e o back-end na Heroku.
O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
